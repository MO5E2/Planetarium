PLANETARIUM\n
An interactive and educational 3D web app that allows users to explore an orbital view of each of the planets in our solar system and an overview of their temperature and size.

PURPOSE
The purpose of the project is to practice building Single-Page React Apps and styling with Tailwind CSS.

TECH STACK
React
Tailwind CSS
Three.js
React Three Fiber
Drei

INSTALLATION
git clone https://github.com/MO5E2/Planetarium.git
cd Planetarium
npm install
npm run dev

SOURCES
Planet image textures are from solarsystemscope.com
All temperature and size information are from science.nasa.gov
